# 00 学习路线

本目录按 17 章路线推进。当前策略是先逐章把 17 章 Markdown 笔记搭好；用户说“继续”就进入下一章。每章末尾的问题先作为后续复习题保留，不阻塞下一章创建。

## 第 1 章：LLM 推理全景图

学习 Prompt、Tokenizer、Token、Embedding、Decoder-only Transformer、Logits、Sampling、自回归生成、Prefill 和 Decode 的位置，以及一次请求从输入到输出的整体链路。

目标：先建立全局地图，不展开复杂公式。

## 第 2 章：Token、Embedding 与 Tensor Shape

学习 Tokenizer、Vocabulary、Token ID、Embedding、Batch Size、Sequence Length、Hidden Size，以及 `[B, S, H]` 的含义。

## 第 3 章：线性层与参数计算

学习 Linear Projection、Weight、Activation、GEMM、GEMV、参数量、FLOPs 和模型权重显存。

## 第 4 章：Self-Attention

学习 Query、Key、Value、`QK^T`、Scale、Softmax、Attention Weight、Causal Mask 和 Attention 输出。必须解释每一步 Tensor Shape。

## 第 5 章：Multi-Head Attention 与 Decoder Block

学习 Head、Multi-Head Attention、Head 拆分和合并、Output Projection、Residual Connection、RMSNorm、FFN、SwiGLU，以及一个现代 Decoder Block 的完整结构。

## 第 6 章：训练与推理的区别

学习 Teacher Forcing、训练阶段的序列并行、自回归推理、为什么推理不能一次生成全部 Token，以及为什么后一个 Token 依赖前一个 Token。

## 第 7 章：Prefill 与 Decode

学习 Prefill、Decode、两阶段输入 Shape、并行性、GEMM 与 GEMV 特征、Compute-bound、Memory-bound、TTFT、TPOT、ITL。

## 第 8 章：KV Cache

学习为什么缓存 K 和 V、为什么不缓存历史 Q、每层 KV Cache、KV Cache Shape、显存估算公式、上下文长度和并发的影响。

## 第 9 章：MHA、MQA、GQA

学习 MHA、MQA、GQA、Q Head、KV Head、为什么 GQA 能减少 KV Cache，以及不同结构的性能取舍。

## 第 10 章：FlashAttention

学习标准 Attention 的中间矩阵、HBM、SRAM、IO、Tiling、Online Softmax、为什么 FlashAttention 是精确算法，以及为什么它主要减少 IO，而不是简单减少理论计算量。

## 第 11 章：PagedAttention

学习 KV Cache 连续分配问题、Block、Logical Block、Physical Block、Block Table、内部碎片、外部碎片、Prefix Caching、Copy-on-Write。

## 第 12 章：Continuous Batching 与 Scheduler

学习 Static Batching、Continuous Batching、Waiting/Running/Finished 请求、Prefill 与 Decode 调度、Chunked Prefill、抢占、TTFT 与吞吐之间的权衡。

## 第 13 章：vLLM Runtime

学习 API Server、Request Processor、Scheduler、KV Cache Manager、Model Runner、Worker、Attention Backend、Streaming Output，以及请求从进入服务到输出 Token 的完整调用路径。

必须以当前官方文档和源码为准。涉及版本变化时必须标注查询日期和版本。

## 第 14 章：量化

学习 FP32、FP16、BF16、FP8、INT8、INT4、W8A8、W4A16、Weight-only Quantization、Activation Quantization、KV Cache Quantization、Per-Tensor、Per-Channel、Per-Group，以及为什么模型变小不等于一定变快。

## 第 15 章：GPU 与 CUDA 推理常识

只讲概念，不运行程序。学习 Grid、Block、Thread、Warp、SM、Register、Shared Memory、Global Memory、Coalesced Access、Bank Conflict、Warp Divergence、Occupancy、Kernel Launch Overhead、Tensor Core 和算子融合。

## 第 16 章：推理性能指标与分析

学习 Latency、Throughput、TTFT、TPOT、ITL、P50、P95、P99、Goodput、SLO、Warmup、正确计时、单请求吞吐与系统吞吐的区别。

## 第 17 章：推理优化总图

学习 FlashAttention、PagedAttention、Continuous Batching、Chunked Prefill、Prefix Caching、CUDA Graph、Quantization、Speculative Decoding、Tensor Parallelism、Prefill-Decode Disaggregation。

明确每项技术解决的是计算问题、显存问题、内存 IO 问题、调度问题、通信问题还是服务延迟问题。
